Loopy for macOS

Local coding loops with a cute co-pilot.

Loopy is the desktop app for the `9xf` engine. It stays local-first, keeps setup light, supports Ollama for on-device models, and lets users bring their own hosted-model API keys without creating an account.

Download latest Docs Release notes

What users get

A native macOS app, a mascot-led first-run setup, a run transcript, a live diff register, and local project storage.

Setup model options

Use Ollama locally or paste a hosted-model API key that stays on the device.

Minimal sign-up

No email collection. No cloud account. No remote execution service required.

1. Install

Download the latest macOS release artifact and open Loopy.

2. Get set up

Loopy helps users connect to http://localhost:11434 or save an API key locally.

3. Start a project

Pick a folder, write a goal, and let the loop plan, build, test, and commit.
Desktop app branding: Loopy. Engine and config compatibility: `9xf`.
Suggested deployment: static site on Vercel/Netlify/GitHub Pages, binaries on GitHub Releases.
